# Email Extraction Automation Script

## Project Overview
This project is a Python automation script developed as part of my internship task.

The script automatically extracts valid email addresses from a text file and saves them into a separate file.

## Features
- Extracts valid emails using Regex
- Removes duplicate email addresses
- Saves results into a new file
- Reduces manual effort in data extraction

## Technologies Used
- Python
- Regex (re module)
- File Handling

## How to Run
1. Install Python
2. Run the script:
   python email_extractor.py

## Output
The program generates a file:
extracted_emails.txt
containing all unique valid emails.
